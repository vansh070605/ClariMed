# Screenshots Directory

This directory is intended to hold visual assets of the ClariMed application for the `README.md`.

## Instructions for Capturing

Please run the Next.js frontend (`npm run dev`), walk through the application, and capture the following screenshots. Save them in this directory with the exact filenames below:

1. `login.png` - The authentication screen.
2. `upload.png` - The drag-and-drop PDF upload interface.
3. `report-details.png` - The structured measurements table and LLM Patient Summary.
4. `dashboard.png` - The high-level KPI dashboard.
5. `trends.png` - The longitudinal Recharts line graph.

*(Once added, these will automatically be referenced by any presentation materials).*
